## What are Compount Components?

```tsx
export default function App() {
  return (
    <Dropdown>
      <Dropdown.Trigger>Dropdown</Dropdown.Trigger>
      <Dropdown.Items>
        <Dropdown.Item>John Soe</Dropdown.Item>
        <Dropdown.Item>Susan Storm</Dropdown.Item>
      </Dropdown.Items>
    </Dropdown>
  );
}
```
